# Windows 11 Dev Workstation Setup (VS Code + WSL Ubuntu + Git + Dev Containers)

Este repositorio documenta, paso a paso, cómo preparar desde cero una estación de trabajo en Windows 11 para desarrollo usando:

- Windows 11 actualizado y configurado
- WSL2 con Ubuntu
- Visual Studio Code y extensiones remotas
- Git y GitHub (PAT o SSH)
- Docker Desktop con integración WSL2
- Dev Containers (VS Code)

Contenido:

1. docs/01-windows-setup.md – Preparar Windows 11, habilitar WSL2, instalar Ubuntu, VS Code y Docker Desktop
2. docs/02-vscode-setup.md – Configurar VS Code y extensiones clave
3. docs/03-wsl-setup.md – Configurar Ubuntu (WSL), paquetes y Git básico
4. docs/04-git-setup.md – Autenticación con GitHub (PAT o SSH) y buenas prácticas
5. docs/05-devcontainers-setup.md – Dev Containers: requisitos, plantilla y uso
6. docs/06-docker-setup.md – Docker Desktop + WSL2: configuración
7. docs/troubleshooting.md – Problemas comunes y soluciones

Uso rápido:

- Revisa los documentos en el orden indicado.
- Opcional: abre este repo en un Dev Container (incluye linters y tooling para docs).
